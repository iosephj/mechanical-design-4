## Guía 2: Metrología básica para taller

<br>

### 1. Medición y errores

Medir es comparar una magnitud con una unidad patrón para saber "cuánto tiene" algo. Por ejemplo comparar el largo de una pieza con milímetros.

Toda medición tiene cierto error. No existe medición perfectamente exacta. Los errores pueden aparecer por:

* **El instrumento:** proviene del desgaste, mala calibración o poca precisión.
* **El operador:** proviene del uso incorrecto como mirar mal la escala, apretar demasiado el micrómetro, medir inclinado.
* **El ambiente:** este puede modificar las medidas, por ejemplo la temperatura alta, la suciedad, las vibraciones, etc.

Para reducir errores hay que:

* usar el instrumento adecuado,
* mantener limpios pieza e instrumento,
* medir varias veces,
* no golpear los instrumentos,
* guardar correctamente,
* verificar el “cero” antes de medir.

**Actividad:**

Escribe la siguiente frase en inglés que resume todo lo anterior, completa los espacios con una sola palabra.

*Measurement _____ comparing a quantity with a standard unit. Every measurement has some error. Errors can come _____ the instrument, the operator or the _____. To reduce _____, use the correct instrument, keep everything clean and check the zero before measuring.

Vocabulary:

* quantity = magnitud
* standard unit = unidad patrón
* environment = ambiente
* clean = limpio
* measuring = medir


<br>

### 2. Tipos de instrumentos de medición

Hay de varios tipos y se pueden clasificar como:

- **Instrumentos para longitud:** Sirven para medir dimensiones lineales como la regla, el calibre Vernier o pie de rey, el micrómetro, etc.
.
- **Instrumentos para ángulos:** Sirven para medir inclinaciones o ángulos como el transportador, las escuadras, el goniómetro.
- **Instrumentos de comparación:** No muestran directamente la medida; comparan diferencias. Ejemplo: el reloj comparador que se usa por ejemplo para centrar piezas en el plato del torno y verificar que giren sin excentricidad.
- **Instrumentos de control:** Sirven para verificar si una pieza cumple una medida. Ejemplos: calibres pasa/no pasa, galgas (el peine usado para controlar el paso de una rosca es una galga), etc.

**Actividad:** 

**a)** Pide un comparador de reloj y dibújalo esquemáticamente señalando: aguja indicadora, punta palpadora, sistema de fijación magnético.

**b)** Dibuja esquemáticamente una rosca triangular métrica y una witworth señala el paso y el ángulo de cada una. Pide una galga para controlar el paso de roscas, busca y pon desde que valor a que valor van los peines de medición, tanto para rosca Witworth (hilos por pulgada) como métrica (donde el paso se mide en milímetros).
